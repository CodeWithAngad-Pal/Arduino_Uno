| **Component**    | **Arduino Pin** | **Description**               |
|:------------------|:---------------:|-------------------------------:|
| Relay Module      |       7         | Controls the relay (load ON/OFF) |
| Power Source      |   VCC, GND      | Provides power to the relay module |
| Load (e.g., bulb) |   Relay NO/NC   | Connects to the relay's output |
