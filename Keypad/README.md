# Arduino Keypad Communication (4x4 Keypad)

## Components:
- **Arduino Uno** (or any other compatible board)
- **4x4 Keypad**
- **Jumper Wires**
- **Breadboard** (optional)

## Circuit Connections:

| Keypad Pin     | Arduino Pin |
|----------------|-------------|
| **Keypad Pin 1** | **Pin 9**   |
| **Keypad Pin 2** | **Pin 8**   |
| **Keypad Pin 3** | **Pin 7**   |
| **Keypad Pin 4** | **Pin 6**   |
| **Keypad Pin 5** | **Pin 5**   |
| **Keypad Pin 6** | **Pin 4**   |
| **Keypad Pin 7** | **Pin 3**   |
| **Keypad Pin 8** | **Pin 2**   |

### Steps to Connect:
1. Connect the **columns** (Pins 1 to 4) of the keypad to the digital pins of the Arduino (Pins 9 to 6).
2. Connect the **rows** (Pins 5 to 8) of the keypad to the digital pins of the Arduino (Pins 5 to 2).
